Install the Grizzly Parrot Trading App

Same site. Faster, cleaner, and pinned on your home screen like a real app.

What this “app” actually is

This is a Progressive Web App (PWA). It’s the exact same content as grizzlyparrottrading.com, just wrapped so your phone can install it:

  • No app store, no account, no extra logins.
  • Loads faster after the first visit because key files are cached.
  • Lives on your home screen like any other app.

Why bother installing it?

  • Speed: Key pages and assets load from local cache after your first visit.
  • Convenience: Tap the icon, not a bookmark, no typing URLs.
  • Focus: Opens in an app-style window (no address bar clutter).

How to install on Android (Chrome)

  1. Open https://grizzlyparrottrading.com/ in Chrome.
  2. Wait a few seconds on the homepage. If the install banner pops up, tap Install.
  3. If you don’t see a banner:
    1. Tap the ⋮ menu (top-right in Chrome).
    2. Tap Install app or Add to Home screen.
    3. Confirm the name Grizzly Parrot Trading and tap Install.
  4. Find the new icon on your home screen or app drawer and open it like any normal app.

How to install on desktop (Chrome / Edge)

  1. Open https://grizzlyparrottrading.com/ in Chrome or Edge.
  2. Look in the address bar for an icon that looks like a little monitor with a down arrow, or:
    • Chrome: ⋮ menu → Install Grizzly Parrot Trading.
    • Edge: ⋯ menu → Apps → Install this site as an app.
  3. Click Install. It will open in its own window like a native app.

What about offline?

After the first load, the app caches core files so the homepage and some assets still work with a weak or dead connection. If you go fully offline and click deep into pages that aren’t cached yet, expect some things not to load. That’s normal for now. Offline behavior will get smarter over time.

Ads and tracking

Ads only load when you have an internet connection. No connection = no ad request. When you’re online, the app uses the same ads and analytics as the normal website.

Already installed?

If you already see the Grizzly Parrot Trading icon on your home screen or app list, you’re done. Open that instead of the browser and use it like any other app.
