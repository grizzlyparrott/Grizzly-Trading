Install the Grizzly Parrot Trading App

Add this site to your phone or desktop and open it like a real app — faster access, cleaner layout, no URL typing.

What You’re Installing

This is a Progressive Web App (PWA). It’s the same Grizzly Parrot Trading site, just installed so you can open it from an icon.

  • No app store required
  • Opens in its own window
  • Loads faster after the first visit

How to Install on Android (Chrome)

  1. Open grizzlyparrottrading.com in Chrome.
  2. Wait a moment — if the install banner appears, tap Install.
  3. If no banner:
    1. Tap the ⋮ menu.
    2. Select Install app or Add to Home screen.
    3. Confirm the name Grizzly Parrot Trading.

How to Install on Desktop (Chrome or Edge)

  1. Open the site in Chrome or Edge.
  2. Look for the install icon in the address bar.
  3. If you don't see it:
    • Chrome: Menu → Install Grizzly Parrot Trading
    • Edge: Menu → Apps → Install this site as an app

Already Installed?

If the icon is already on your home screen or app list, you're done — open it from there anytime.
