Tradovate Order Presets & Auto Brackets

Set default quantity, stops, targets, break-even behavior, trailing rules, and auto-brackets for every trade you place.

Where Order Presets Are Located

All presets live in this menu:

  1. Open the left sidebar.
  2. Click Application Settings.
  3. Select Orders.

This is different from general order entry covered in Tradovate Order Entry.

Setting a Default Quantity

The simplest preset is your auto-quantity:

  1. Under Orders, find Default Quantity.
  2. Choose 1, 2, 3, etc.

Configure Auto Brackets

Auto brackets are automatically placed take-profit and stop-loss orders.

To enable brackets:

  1. Application Settings → Orders
  2. Turn on Use Default Bracket

Set your offsets:

  • Target Offset (profit distance)
  • Stop Offset (stop distance)

These work perfectly with your simple layout from Simple Chart Layout.

Break-Even Automation

Tradovate lets you auto-move stops to break-even:

  • BreakEven Trigger — how many ticks in profit must price go
  • BreakEven Offset — how many ticks above/below break-even to place the stop

Trailing Stop Rules

You can automate trailing logic:

  • Trail Trigger — when the trailing behavior activates
  • Trail Amount — distance price trails behind
  • Trail Step — how often the stop updates

Saving Multiple Presets

You can create multiple bracket configurations:

  1. Go to Orders → Saved Presets.
  2. Click Create New.
  3. Name them logically:
    • scalp-bracket
    • runner-bracket
    • big-stop-setup

Apply Presets to DOM or Chart

You can force a specific preset to always load for a specific interface:

  • Chart Trading
  • DOM
  • Order Ticket

Combine this with your DOM understanding from Tradovate DOM Settings.

Common Problems

  • Preset applies on chart but not DOM → fix in individual DOM settings
  • Preset doesn’t load → “Use Default Bracket” not turned on
  • Stops/targets don’t show → using SIM with incorrect offsets
  • Not enough margin → preset target/stop too large
