Bookmap Built-In Alerts: Liquidity, Delta, and Volume Signals

Bookmap has a set of built-in alerts that trigger when important order flow conditions occur. These aren’t sweep alerts or big add-ons—these are the basic platform notifications that tell you when liquidity appears, disappears, when delta surges, or when trade speed spikes. If you use Bookmap intraday, these alerts save you from staring at every tick.

What Bookmap Alerts Actually Do

Built-in alerts monitor core events:

  • liquidity changes
  • delta thresholds
  • volume-speed bursts
  • price reaching important liquidity
  • custom sound triggers

They help you catch moves early without watching every microsecond of order flow.

1. Liquidity Appearance Alerts

These trigger when a large order suddenly appears in the book. Bookmap highlights it and plays a sound if enabled.

Use this to catch:

  • spoofers dropping big resting orders
  • real liquidity defenders stepping in
  • big players preparing to absorb or reverse price

2. Liquidity Disappearance Alerts

These alerts fire when a large order is suddenly removed—often seconds before a breakout.

This can signal:

  • liquidity being pulled to let price run
  • a stop run about to fire
  • a fake wall disappearing

Pair this alert with the heatmap if you want a clean breakout read.

3. Delta Threshold Alerts

Bookmap can alert you when delta hits a certain value or when it flips aggressively.

These tell you:

  • buyers overwhelming sellers
  • sellers overwhelming buyers
  • trend confirmation
  • possible reversal from exhaustion

For deeper understanding, see Bookmap CVD Explained.

4. Volume Speed Alerts

These fire when the trade rate spikes above a set threshold. They expose:

  • momentum bursts
  • sweeps starting
  • stop cascades kicking off
  • algorithmic acceleration

These alerts catch moves you’d miss with candles.

5. Price-Level Hit Alerts

You can set Bookmap to alert when price reaches specific areas of interest:

  • major liquidity bands
  • imbalances
  • high-volume nodes
  • user-defined markers

Saves you from babysitting key levels.

6. Sound Alerts

You can assign sound cues for each alert type:

  • ding for liquidity appearing
  • click for liquidity disappearing
  • sharp tone for speed bursts
  • custom audio for delta events

Once configured, you can trade without staring at the chart constantly.

7. Visual Alerts

Bookmap marks alerts visually with:

  • flashes
  • highlighted liquidity
  • colored markers
  • alert timestamps

These make it easy to review order flow after the fact.

Final Thoughts

Bookmap’s built-in alerts give you a heads-up on critical order flow events without needing add-ons or plugins. Liquidity changes, delta shifts, and volume bursts tell you when real activity is happening. Set them correctly and you’ll react faster without staring at every pixel all day.
