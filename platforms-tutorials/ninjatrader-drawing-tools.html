How to Use Drawing Tools in NinjaTrader

Learn how to mark levels, draw trendlines, create zones, measure legs, and clean up your chart in NinjaTrader using its built-in drawing tools.

Where Drawing Tools Are Located

NinjaTrader hides drawing tools behind a right-click menu. To access them:

  1. Right-click directly on your chart.
  2. Hover over Drawing Tools.
  3. Select the tool you want (trendline, horizontal line, Fibonacci, etc.).

Horizontal Lines (Key Levels)

The most common tool. Perfect for marking levels you identified in Market Structure Basics.

  1. Right-click → Drawing Tools → Horizontal Line.
  2. Click the price level where you want the line.
  3. Right-click the line to edit color, thickness, or label.

Trendlines

Trendlines help you visualize swings, breaks, and momentum shifts.

  1. Right-click → Drawing Tools → Trend Line.
  2. Click swing low → drag to a later swing low.
  3. Adjust thickness or color in the properties if needed.

Rectangles / Zones

Great for drawing consolidation areas, demand zones, and supply zones.

  1. Right-click → Drawing Tools → Rectangle.
  2. Click and drag to create your zone.
  3. Edit transparency so you can see candles underneath.

Rulers (Measure Price + Time)

The ruler tool measures:

  • Tick distance
  • Point distance
  • Time duration
  • Price difference

To use:

  1. Right-click → Drawing Tools → Ruler.
  2. Click start point → click end point.
  3. Look at the small metrics box for details.

Adjusting and Removing Drawings

  • Click any drawing to select it.
  • Drag to move.
  • Right-click → Remove to delete.
  • Right-click → Properties to edit color/size/text.

Saving Drawings in Your Workspace

Drawing tools are saved inside your workspace. Use: NinjaTrader Workspaces to lock in your layout.
