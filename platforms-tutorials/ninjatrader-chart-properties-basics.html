NinjaTrader Chart Properties Basics

The Chart Properties window controls every visual and behavioral setting on your NinjaTrader chart. If your chart looks messy, scrolls weird, or feels off, this is where you fix it. Most traders ignore these settings and live with sloppy charts. This guide shows you how to dial everything in cleanly.

How to Open Chart Properties

Right-click on the chart → Properties…

This opens the full Chart Properties panel.

Background and Color Settings

These settings control the visuals:

  • Background – main chart color
  • Grid lines – horizontal/vertical grid colors
  • Axis text – price and time labels
  • Crosshair – color and style

If your chart ever becomes unreadable, it’s usually because these colors were changed unintentionally.

Grid Line Control

Grid lines help or hurt depending on your style:

  • Show horizontal grid
  • Show vertical grid

Most futures traders turn off vertical grid lines to clean up the view.

Scale Settings (Important)

This controls how the Y-axis behaves:

  • Right scale only
  • Left scale only
  • Both
  • Auto scale – chart always fits price action
  • Fixed scale – manual zoom control

If your chart keeps “jumping,” auto scale is usually turned on.

Session Break Lines

These add markers between trading sessions.

Enable or disable with:

  • Show session break line

Useful if you're studying overnight vs RTH behavior.

Crosshair Settings

  • Style: pointer, crosshair, global crosshair
  • Color: customize for visibility

Global crosshair lets you sync the crosshair across all charts.

You cover crosshair behavior in depth here: NinjaTrader Crosshair Basics

Data Box Settings

The Data Box shows indicator values for the bar you're pointing at.

  • Show Data Box
  • Data Box font/size

Most traders keep it off unless doing deep analysis.

Chart Behavior Settings

This section controls how the chart moves when you scroll:

  • Auto scale
  • Scroll bars
  • Center price on scale
  • Always show last price marker

If your chart suddenly stops following the price, disable “Fixed scale” or re-enable “Auto scale.”

Trading Visuals

This affects how orders and executions appear:

  • Plot executions – show fill markers
  • Plot trade markers – show entry/exit tags
  • Order markers – show working orders

Scalpers often hide execution markers to reduce clutter.

Final Thoughts

The Chart Properties window controls the readability of your entire workspace. If your chart looks chaotic or scrolls unpredictably, it's almost always because a Chart Properties option is set wrong. Learn this window once and you’ll avoid 90% of chart frustration going forward.
