Tradovate Data & Connection Setup

How to connect Tradovate to live data, SIM data, prop firm data, and how to fix the most common “price not moving” connection problems.

Where Data Settings Are Located

All data connections in Tradovate are controlled from:

  1. Sidebar → Application Settings
  2. Click Data

There’s also a connection status indicator in the top-right of the app.

SIM Data (Free)

Tradovate provides free SIM data with a few limitations:

  • It’s delayed during overnight hours
  • It doesn’t include all historical tick data
  • You need to be connected to the Demo account

How to use it:

  1. Open account dropdown.
  2. Select Demo.
  3. Make sure Simulation Server is connected.

Live Data (Ninja / Tradovate Brokerage)

If you have a live Tradovate or NinjaTrader brokerage account:

  1. Go to Data → Market Data Subscriptions.
  2. Enable CME / NQ / CL / whatever you trade.
  3. Switch account dropdown to your live trading account.
  4. Confirm green connection icon.

Prop Firm Connections (Apex, Leeloo, Topstep, etc.)

Tradovate can connect to most prop firms using Rithmic:

  1. Sidebar → Application Settings.
  2. Click Connections.
  3. Add a Rithmic connection.
  4. Enter your Rithmic username and password.
  5. Select the correct gateway (often Rithmic 01).

This pairs perfectly with: NinjaTrader Data Feed Setup .

Why Price Might Not Be Moving

Most common causes:

  • No market hours (weekend, holiday, maintenance)
  • SIM data is delayed
  • Rithmic login error
  • You’re subscribed only to indices but trading CL or GC
  • Your prop account is inactive or expired

How to Check Connection Status

The top-right connection indicator:

  • Green = fully connected
  • Yellow = connecting
  • Red = disconnected

Data Settings to Review

  • Subscribe to Level 1 (required for charts)
  • Subscribe to Level 2 (optional for DOM depth)
  • Enable mobile data access if using the app
