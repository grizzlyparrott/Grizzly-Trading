NinjaTrader Chart Linking

Use NinjaTrader's linking system to sync instruments, timeframes, and drawings across multiple charts and windows.

What Chart Linking Does

Chart linking lets you:

  • Change the instrument on one chart and update others automatically
  • Sync timeframes across multiple charts
  • Optionally share drawings between charts

Linking sits on top of your normal layout from NinjaTrader Workspaces.

Link Colors and Link Types

Each chart has a small colored box in the toolbar:

  • The color identifies a link group.
  • Charts with the same color are linked together.
  • You can link by instrument, interval, or both.

How to Link Charts by Instrument

  1. Open two or more charts.
  2. On the first chart, click the link color icon and choose a color.
  3. Select link mode: Instrument (or Instrument + Interval).
  4. On the second chart, choose the same color and link mode.

Now, changing ES → NQ on one chart changes all charts in that link group.

Linking Timeframes (Intervals)

You can link intervals as well:

  • Instrument only: symbol syncs, timeframe stays independent.
  • Interval only: timeframe syncs, symbol stays independent.
  • Instrument & Interval: both sync together.

Most traders use instrument-only links and keep timeframes different (for example 1-minute and 5-minute), like discussed in Execution vs Idea.

Sharing Drawings Between Charts

Drawings can be:

  • Visible on the current chart only
  • Visible on all charts of the same instrument

Adjust this in each drawing’s properties. Use this together with NinjaTrader Drawing Tools.

Linking DOMs and Other Windows

You can also link:

  • Charts and DOMs for the same instrument
  • SuperDOM with the main chart
  • Market Analyzer windows

Use the same color link on each window so changing the instrument in one updates all linked windows, including the DOM from NinjaTrader DOM Settings.

Common Mistakes

  • Linking all charts together when you wanted only one group
  • Changing the wrong chart and flipping every window to a new instrument
  • Accidentally linking by interval and wondering why timeframes keep changing

Recommended Linking Setup

Simple, low-confusion approach:

  • One color group per product (index futures, oil, etc.)
  • Instrument-only links for charts with different timeframes
  • DOM linked to its main execution chart
