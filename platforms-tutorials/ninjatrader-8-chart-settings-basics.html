NinjaTrader 8 Chart Settings Basics

NinjaTrader 8 gives you extremely customizable charts. That’s good, but it also means beginners get lost in the menus. This guide strips out the noise and shows you the basic settings you actually need to control—data series, bar types, scaling, colors, trading display, and templates.

Opening Chart Settings

Right-click anywhere on a chart → Data Series… or Properties…. These two menus control 90% of the chart behavior you care about.

If you can’t find a setting, it’s almost always in one of these two windows.

Data Series: Where the Real Chart Settings Live

Data Series is the most important settings window in NinjaTrader 8. This is where you choose:

  • Bar type (minute, tick, volume, range, Heiken Ashi, etc.)
  • Bars value (5-minute, 1500-tick, 10-range, etc.)
  • Trading hours template
  • Candle colors
  • Borders, wicks, plot thickness

Most traders use:

  • 5-Minute
  • 1-Minute
  • Tick charts like 1500 or 1000
  • Range charts

Minute charts update on time. Tick charts update on volume. Range charts update on movement.

Chart Properties: Layout and Behavior

Right-click → Properties…

This window controls:

  • Grid lines
  • Background color
  • Price scale
  • Candle spacing
  • Crosshair style
  • Session break lines

If your chart looks “weird,” too zoomed, or too spaced out, Properties is the fix.

Scaling: How to Stop the Chart From Jumping

NinjaTrader 8 supports three major scale modes:

  • Automatic – chart adjusts to show all price action
  • Fixed – chart stays locked to set values
  • Fixed + Auto – mix of the two

If your chart keeps snapping up and down when price moves, switch to: Right-click → Properties → Scale Justify = Right

Or scroll the price axis until NT8 offers a reset option.

How to Change Candle Colors

Right-click chart → Data Series…

  • Bar Color Up
  • Bar Color Down
  • Candle Outline
  • Wick Color

Many traders pick:

  • Green up / Red down (classic)
  • White up / Red down (dark mode)

Whatever you choose, save it as a template.

Templates: Save Time and Stop Repeating Settings

Once your chart looks the way you want:

  1. Right-click chart
  2. Click Templates → Save
  3. Give it a name like “MyBasicChart”

Next time you open a chart:

  1. Right-click
  2. Templates → Load

Done. No more repeating settings.

Performance Settings: Keep NT8 Fast

Some chart configurations slow down NinjaTrader:

  • Huge historical lookback
  • High-tick charts (e.g., 5000-tick)
  • Tons of indicators
  • Multiple charts all pulling deep data

To avoid lag:

  • Limit days to load
  • Use ETH or RTH templates with fewer sessions
  • Only keep indicators you actually use

NinjaTrader 8 is fast, but not magic. Don’t overload it with nonsense.

Final Thoughts

NinjaTrader 8 chart settings are deep, but once you understand Data Series, Properties, scaling, and templates, the platform becomes predictable instead of chaotic. These are the core tools you’ll use every day—master them first before you worry about indicators or automation.
