How to Save and Load Workspaces in NinjaTrader

Lock in a clean layout once and stop rebuilding charts and DOM windows every time you open NinjaTrader.

What a Workspace Is

In NinjaTrader, a workspace is just your full layout: charts, DOMs, watchlists, indicators, and how everything is arranged on the screen. Saving a workspace means you can close NinjaTrader and come back to the same setup later.

It pairs perfectly with a simple chart layout like the one in Simple, Readable Chart Layout .

Creating a Clean Workspace First

Before saving anything, get your layout sane:

  • One or two charts with the instruments you actually trade.
  • One DOM window for your main contract.
  • Chart Trader or DOM order entry visible, not both stacked on each other.
  • Indicators you actually use, nothing extra.

If you have not done the basics yet, start with NinjaTrader Basics and then come back to lock in the layout.

How to Save a Workspace in NinjaTrader

  1. Arrange your charts, DOMs, and other windows the way you like.
  2. Go to the main NinjaTrader control center.
  3. Click Workspaces.
  4. Choose Save Workspace As....
  5. Give it a clear name like Futures-ES-main or Prop-Eval-Layout.
  6. Click OK to save.

Use different workspace names for different jobs. For example, one layout for your main trading, another layout for testing or replay.

How to Load a Workspace

Loading it is even simpler:

  • Open the NinjaTrader control center.
  • Click Workspaces.
  • Click the name of the workspace you want to load.
  • Wait a moment while all the charts and DOMs open in the saved positions.

Make sure the instruments and account (SIM vs live) match what you expect before you start clicking orders. That ties back to the execution focus in Execution vs Idea .

Renaming and Closing Workspaces

To clean things up:

  • Use Workspaces → Save Workspace As... to create a renamed copy.
  • Use Workspaces → Close Workspace to close ones you do not use.
  • Keep only a few active to avoid NinjaTrader loading a ton of windows at startup.

Workspaces and Order Entry

Your workspace should match the way you enter trades:

  • If you trade from the chart, make sure Chart Trader is configured like in How to Place Orders in NinjaTrader .
  • If you trade from the DOM, double check the DOM layout you use along with DOM Trading Basics .

Backing Up Your Workspace

NinjaTrader has built-in backup options. Use them if you are happy with your setup:

  • From the control center, open the backup/restore menu.
  • Create a backup that includes workspaces and templates.
  • Store the backup file somewhere you can find it later.

Having a backup matters more if you use multiple prop firm logins and layouts like the ones discussed in How Prop Firm Evaluations Work .
