6M Tick Size Explained

The 6M contract moves in increments of 0.0001, and each tick is worth roughly $2.50–$3.00 depending on the USD/MXN exchange rate. If you don’t fully understand this math, you’re trading blind. This breakdown shows exactly how every price move turns into real P&L.

The Exact Tick Structure

6M quotes like this:

19.5650
19.5651
19.5652

Each step is 0.0001. That’s the minimum price fluctuation. Multiply it by the contract size and you get the tick value.

Tick Value Formula

Here’s the blunt math:

Tick Value = Contract Size × Tick Size
Tick Value = 500,000 MXN × 0.0001
Tick Value = 50 MXN

You then convert that 50 MXN to USD:

Tick Value (USD) = 50 ÷ Exchange Rate

So at different USD/MXN levels, the tick value changes slightly. Here's the chart.

This is why 6M can melt your account—it’s not a static $1.25 per tick like many micros. It hits harder the stronger MXN is.

Most beginners underestimate how big 0.0100 is. That’s 100 ticks. Multiply by ~$2.70 per tick and you’re staring at ~$270 per contract. And that’s a normal day.

That volatility is why prop firms allow 6M—it provides massive movement for low margin.

Real Example: A 6M Move Converted to Dollars

Let’s say price jumps from 19.4200 to 19.4550. That’s a 0.0350 move.

0.0350 ÷ 0.0001 = 350 ticks
350 ticks × $2.60 = $910 profit/loss

You blink wrong and that’s a mortgage payment.

Why 6M Tick Math Matters

You never set stops in dollars first. You set them in ticks based on ATR. 6M can swing 80–120 ticks in a heartbeat. Using a 20–30 tick stop like you would in 6E or 6J is suicide. Respect the tick math or you get folded immediately.

The Bottom Line

Every tick in 6M is worth around $2.50–$3.00, and the contract routinely moves hundreds of ticks per session. If you don’t understand tick math, you can’t size your trades, manage risk, or even survive volatile peso conditions.
